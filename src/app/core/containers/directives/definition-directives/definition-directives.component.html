¿QUE ES UNA DIRECTIVA?

En angular las directivas son clases que añaden comportamientos a elementos html, los cuales se ejecutan sobre el DOM (Document Object Model).

Con las directivas podremos hacer lo siguiente:

  • Modificar la estructura del DOM añadiendo/manipulando o eliminando directamente desde la vista HTML.
  • Modificar la apariencia general de un elemento agregando/eliminando clases o una propiedad en concreto.

Angular por defecto nos brinda una serie de directivas listas para ser utilizadas las cuales son NgClass, NgStyle, *ngIf, *ngFor, *ngSwitch, etc. Sin embargo Angular ofrece la posibilidad de crear directivas personalizadas según la necesidad que se tenga.
