¿Qué es una directiva?

En angular las directivas son clases que añaden comportamientos a elementos html, los cuales se ejecutan sobre el DOM (Document Object Model).

Con las directivas podremos modificar la estructura del DOM añadiendo/manipulando o eliminando directamente desde la vista HTML y modificar la apariencia general de un elemento agragando/eliminando clases o una propiedad en concreto.
